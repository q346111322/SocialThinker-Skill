# Max Weber 马克斯·韦伯 — 主要著作与思想参考

## 一、《新教伦理与资本主义精神》(1905)

### 核心论点
加尔文宗教教义（预定论、天职观）无意间培养了一种系统化的生活方式和资本积累的心态，这构成了近代资本主义精神的文化基础。

### 关键概念
- **天职观 (Beruf/Calling)**：世俗劳动被视为神圣使命
- **内在的世俗禁欲主义**：节制消费、勤勉劳动、系统化理性生活
- **预定论 (Predestination)**：信徒通过世俗成功寻求"被选"的确证

### 现代应用
- 分析不同文化价值观对经济发展的影响
- 理解硅谷"工作伦理"的文化根源
- 评估"996文化"等当代工作文化现象

## 二、《经济与社会》(1922，遗著)

### 核心内容
社会学概念和方法的百科全书式著作，涵盖权力、支配、组织、法律、城市等几乎所有社会学主题。

### 关键概念
- **社会行动四种类型**：目的理性、价值理性、情感行动、传统行动
- **三种正当支配类型**：传统型、魅力型、法理型
- **科层制 (Bureaucracy)**：现代社会中法理型支配的组织形式

### 现代应用
- 分析现代组织的科层化与效率
- 理解政治领导力的不同基础
- 评估制度设计和组织改革

## 三、韦伯方法论核心

### 理想类型 (Ideal Type)
通过片面强调某些特征构建的分析工具，不是对现实的描述，而是认识现实的手段。

### 价值无涉 (Wertfreiheit)
社会科学研究应区分事实判断和价值判断，研究者应追求客观分析。

### 理解 (Verstehen)
通过理解行动者赋予其行为的主观意义来解释社会现象。

## 分析模板

| 分析维度 | 核心问题 |
|---------|---------|
| 权威类型 | 当前权威基础是传统、魅力还是法理？ |
| 理性化程度 | 形式理性在多大程度上主导？ |
| 社会行动类型 | 行动者的行为是目的理性还是价值理性驱动？ |
| 社会分层 | 阶级、地位、政党三维度的配置？ |
| 铁笼效应 | 理性化是否导致了意义的丧失？ |
