# Ray Dalio 雷·达里奥 — 主要著作与思想参考

## 一、《原则》(2017)

### 核心论点
人生和工作的成功来自于拥有清晰的原则并系统性地遵循它们。痛苦 + 反思 = 进步。

### 关键原则
- **极度透明和极度开放**：欢迎批评，拥抱不同意见
- **可信度加权**：不是所有人的意见同等重要
- **系统化决策**：将原则编码为可重复的决策流程

## 二、《应对大型债务危机的原则》(2018)

### 核心框架
通过研究48+次历史债务危机，达里奥总结出债务危机的共同模式：

### 债务周期模型
1. **早期阶段**：债务增长快于收入，但资产价格上升使感觉良好
2. **泡沫阶段**：债务增长不可持续，资产价格虚高
3. **危机阶段**：资产价格下跌，债务不可偿还
4. **去杠杆阶段**：四种工具的平衡使用
   - 债务削减（违约/重组）
   - 紧缩政策
   - 财富转移
   - 央行印钞

### "优雅的去杠杆" vs "丑陋的去杠杆"
- **优雅的去杠杆**：四种工具的平衡使用，名义增长率>名义利率
- **通缩性去杠杆（丑陋）**：过度紧缩导致螺旋式下降
- **通胀性去杠杆（丑陋）**：过度印钞导致货币贬值

## 三、《原则：应对变化中的世界秩序》(2021)

### 大周期理论
帝国和储备货币遵循约250年的大周期：

### 帝国兴衰的八大指标
1. 教育质量
2. 竞争力/创新能力
3. 经济产出（GDP份额）
4. 全球贸易份额
5. 军事实力
6. 金融中心地位
7. 储备货币地位
8. 技术/科技领先程度

### 六个阶段
1. 新秩序建立（战争/革命后）
2. 资源分配体系建立
3. 和平繁荣期
4. 过度阶段（贫富差距扩大、债务积累）
5. 财务困难与内部冲突
6. 内战/革命 → 新秩序

### 五大力量
1. 债务/货币/资本市场周期
2. 国内秩序/混乱周期
3. 国际秩序/混乱周期
4. 天灾（疫情、气候）
5. 科技/创新

## 四、《国家如何破产》(2025)

### 核心论点
当政府债务增长速度持续超过偿还能力时，国家就会"破产"。这通常表现为货币贬值和通胀而非直接违约。
