# Karl Popper 卡尔·波普尔 — 主要著作与思想参考

## 一、《科学发现的逻辑》(1934/1959)

### 核心论点
科学知识的增长不是通过归纳（从具体到一般）而是通过猜想与反驳：提出大胆假说，然后尝试证伪它们。

### 关键概念
- **可证伪性 (Falsifiability)**：一个理论是科学的，当且仅当它能被经验检验所否证
- **反归纳主义**：观察无论多少都不能"证明"一个普遍理论
- **批判理性主义**：理性的态度是接受批评并从错误中学习

### 现代应用
- 评估经济预测和社会理论的科学性
- 识别不可证伪的意识形态伪装
- 建立更严谨的分析和决策框架

## 二、《开放社会及其敌人》(1945)

### 核心论点
从柏拉图到黑格尔再到马克思，西方哲学传统中有一条"历史决定论"的暗线，它为极权主义提供了知识基础。开放社会必须保持对批评和纠错的开放。

### 关键概念
- **开放社会**：允许批评、实验和纠错的社会
- **封闭社会**：固守传统、不允许质疑的社会
- **历史决定论批判**：没有"历史必然性"，未来是开放的
- **宽容悖论**：无限宽容导致宽容的毁灭

### 现代应用
- 评估民主制度的自我纠错能力
- 分析信息环境对开放社会的影响
- 理解如何在保持开放的同时防御极端主义

## 三、《历史决定论的贫困》(1957)

### 核心论点
社会科学不可能发现"历史规律"来预测社会的未来走向，因为：
1. 人类知识的增长不可预测
2. 知识增长影响社会进程
3. 因此社会进程不可预测

## 四、《猜想与反驳》(1963)

### 核心论点
知识的增长遵循"试错法"——通过大胆的猜想和严格的反驳来推进。

## 波普尔批判思维模板

| 分析维度 | 核心问题 |
|---------|---------|
| 可证伪性 | 这个声明能被什么证据否定？ |
| 替代假说 | 还有什么其他解释？ |
| 反驳尝试 | 最强的反面论证是什么？ |
| 确认偏见 | 我们是否只在寻找支持性证据？ |
| 开放性 | 是否欢迎批评和纠正？ |
| 渐进方案 | 可测试的渐进改善方案是什么？ |
