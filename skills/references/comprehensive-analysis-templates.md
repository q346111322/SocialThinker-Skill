# 综合分析模板 — 多思想家视角分析框架

本文档提供从多个思想家视角综合分析社会、经济和政治问题的结构化模板，以及多 Agent 并行调用 Skill 的规范。

---

## 零、多 Agent 并行分析规范

### 核心模式：每个 Agent 模仿一位思想家，调用对应 Skill 分析

当用户提出需要多维度分析的问题时，按照以下流程执行：

#### 第1步：信息收集
- 使用 `web_search` 收集最新背景信息和数据
- 至少从2个独立来源交叉验证关键事实
- 整理为结构化的"背景概要"

#### 第2步：选择思想家组合
根据问题类型选择最合适的3-8位思想家（参见下方组合推荐表）

#### 第3步：并行启动 Agent
使用 `team_create` + `task()` 为每位思想家启动一个 Agent，每个 Agent：
- **读取对应的 SKILL.md**（获取完整方法论框架）
- **模仿该思想家的思维方式**（严格使用其分析工具）
- **独立分析**（不参考其他思想家的观点）
- **按统一的输出格式**返回分析结果

#### 第4步：综合总结
由主 Agent 收集所有分析结果，按以下结构综合：
1. **共识点**：所有思想家一致同意的判断
2. **核心分歧**：思想家之间的观点对立（用表格呈现）
3. **最终综合判断**：短期/中期/长期预判
4. **最大风险**：多数思想家警告的危险
5. **最大希望**：多数思想家看到的机遇

## 一、世界新闻与时事分析模板

### 使用场景
分析重大国际新闻事件，理解其深层含义和长远影响。

### 分析步骤

#### 第1步：事实确认（波普尔视角）
- 报道的核心事实是什么？
- 信息来源是否可靠？
- 哪些声明是可证伪的事实，哪些是不可检验的观点？
- 是否存在对立信息或反面证据？

#### 第2步：权力分析（福柯 + 马基雅维利视角）
- 谁在这个事件中掌握权力？权力的来源是什么？
- 主流话语是如何框定这个事件的？
- 被遮蔽或沉默的声音是什么？
- 各方的真实利益是什么（剥离修辞后）？

#### 第3步：经济维度分析（斯密 + 凯恩斯 + 哈耶克视角）
- 自由市场视角：市场信号告诉我们什么？（斯密）
- 需求管理视角：对总需求和就业的影响？（凯恩斯）
- 知识问题视角：中央决策是否忽视了分散的地方知识？（哈耶克）
- 三种视角的共识和分歧是什么？

#### 第4步：社会结构分析（韦伯 + 涂尔干视角）
- 权威类型：涉及的权威是传统型、魅力型还是法理型？（韦伯）
- 社会团结：事件是增强还是削弱了社会凝聚力？（涂尔干）
- 理性化：是否涉及理性化/科层化的推进或抵抗？（韦伯）
- 失范：是否存在社会规范崩溃的迹象？（涂尔干）

#### 第5步：政治哲学分析（阿伦特 + 马基雅维利视角）
- 公共领域：这对公共讨论空间有何影响？（阿伦特）
- 极权倾向：是否存在极权主义的倾向或迹象？（阿伦特）
- 战略分析：各方的virtù（能力）和fortuna（机遇）如何？（马基雅维利）
- 平庸之恶：是否有人在"只是服从命令"中参与了伤害？（阿伦特）

#### 第6步：历史周期分析（达里奥视角）
- 大周期定位：这个事件在大周期中的位置是什么？
- 债务/货币影响：对债务周期和货币体系有何影响？
- 内部秩序：对国内政治稳定有何影响？
- 外部秩序：对国际秩序有何影响？
- 历史类比：历史上什么时期最像当前局面？

#### 第7步：综合判断
- 各思想家视角的共识是什么？
- 分歧在哪里？
- 最可能的发展方向是什么？
- 需要关注的风险和机遇是什么？

## 二、经济形势分析模板

### 使用场景
分析宏观经济趋势，评估经济政策，预判经济走向。

### 分析框架

| 维度 | 斯密视角 | 凯恩斯视角 | 哈耶克视角 | 达里奥视角 |
|------|---------|-----------|-----------|-----------|
| 经济增长 | 分工与市场扩大 | 总需求管理 | 知识发现与创新 | 生产力趋势 |
| 通货膨胀 | 货币供给过剩 | 成本推动或需求拉动 | 信贷过度扩张 | 债务周期阶段 |
| 失业 | 劳动力市场障碍 | 有效需求不足 | 不当投资的修正 | 周期性调整 |
| 政策建议 | 减少干预 | 财政和货币刺激 | 健全货币制度 | 美丽的去杠杆 |
| 贸易 | 自由贸易 | 管理贸易 | 开放竞争 | 大国博弈 |

### 深度分析清单

1. **周期定位**（达里奥）
   - [ ] 短期债务周期位置（5-8年）
   - [ ] 长期债务周期位置（50-75年）
   - [ ] 内部秩序周期阶段
   - [ ] 外部秩序周期阶段

2. **市场机制**（斯密 + 哈耶克）
   - [ ] 价格信号是否正常
   - [ ] 市场是否自由运作
   - [ ] 知识问题是否被尊重
   - [ ] 竞争是否充分

3. **需求侧**（凯恩斯）
   - [ ] 总需求各组成部分状态
   - [ ] 动物精神/市场信心
   - [ ] 流动性状况
   - [ ] 乘数效应估计

4. **制度环境**（韦伯 + 涂尔干）
   - [ ] 制度理性化程度
   - [ ] 社会凝聚力状态
   - [ ] 规范体系完整性
   - [ ] 社会信任水平

## 三、社会问题分析模板

### 使用场景
分析社会不平等、文化冲突、社会运动、科技对社会的影响等。

### 多维度分析

#### A. 结构维度
- **社会事实**（涂尔干）：什么外在于个人的力量在起作用？
- **权力/知识**（福柯）：哪些知识体系在定义和管理这个问题？
- **社会分层**（韦伯）：阶级、地位、权力三个维度的配置？

#### B. 行动维度
- **理解社会学**（韦伯）：行动者赋予行为什么主观意义？
- **政治行动**（阿伦特）：是否存在真正的政治行动空间？
- **话语构建**（福柯）：问题是如何被话语构建出来的？

#### C. 规范维度
- **道德情操**（斯密）：公正旁观者会如何评判？
- **社会团结**（涂尔干）：社会团结是否受到威胁？
- **开放社会**（波普尔）：批评和纠错机制是否正常运作？

#### D. 变革维度
- **新起点**（阿伦特）：哪里孕育着新的开始？
- **文化演化**（哈耶克）：规则和制度是否在适应性地进化？
- **渐进改革**（波普尔）：小规模、可检验的改善方案是什么？

## 四、未来走向预判模板

### 使用场景
评估某一趋势的未来发展方向，包括经济趋势、地缘政治走向、技术发展等。

### 预判方法论

#### 第1步：承认预测的局限性
- "我所不知道的比我所知道的要重要得多"（达里奥）
- "对历史走向的预测不可能精确"（波普尔）
- 给出概率而非确定性判断

#### 第2步：多模型思维
同时运用多个思想家的框架分析，看哪些结论是一致的：
- 斯密的市场逻辑
- 凯恩斯的周期逻辑
- 哈耶克的秩序逻辑
- 达里奥的历史周期逻辑
- 韦伯的理性化逻辑

#### 第3步：情景分析
构建至少三种情景：
- **基准情景**（最可能）：多数模型支持的方向
- **乐观情景**：新起点/制度创新/技术突破
- **悲观情景**：周期恶化/制度崩溃/冲突升级

#### 第4步：反驳检验（波普尔）
- 什么证据能否定我们的预测？
- 我们是否在进行确认偏见？
- 最强的反面论证是什么？

#### 第5步：持续监测指标
确定需要持续关注的关键指标：
- 经济指标（达里奥框架）
- 社会指标（涂尔干/韦伯框架）
- 政治指标（阿伦特/马基雅维利框架）
- 权力/话语指标（福柯框架）

## 五、思想家对话模拟

### 使用场景
模拟不同思想家对同一问题的讨论，展现多元视角的碰撞。

### 对话结构

1. **开场**：每位思想家从自己的核心关切出发阐述立场
2. **交锋**：思想家之间的分歧和辩论
3. **融合**：寻找不同视角之间的互补点
4. **结论**：综合各家之长的整合性分析

### 经典辩论对

| 辩论主题 | 对手A | 对手B | 核心分歧 |
|---------|-------|-------|---------|
| 政府干预 | 凯恩斯 | 哈耶克 | 需求管理 vs. 自发秩序 |
| 权力本质 | 阿伦特 | 福柯 | 集体行动 vs. 弥散性权力 |
| 社会方法论 | 韦伯 | 涂尔干 | 理解(verstehen) vs. 实证 |
| 道德与政治 | 斯密 | 马基雅维利 | 道德情操 vs. 政治现实 |
| 预测能力 | 达里奥 | 波普尔 | 模式识别 vs. 可证伪性 |
